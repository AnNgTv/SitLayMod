# Sit and Lay (SitLayMod)

A simple Fabric mod for Minecraft 1.17.1 that allows players to sit and lay down.

## Features

- **Sit on Blocks:** Right-click on any **Slab** or **Stairs** with an empty hand to sit down.
- **Sit on Players:** Sneak and right-click on another player with an empty hand to sit on them.
- **Commands:**
  - `/sit`: Sit down wherever you are.
  - `/lay`: Lay down wherever you are.
- **Automatic Cleanup:** Seats (invisible armor stands) are automatically removed when you stand up or disconnect.

## Installation

1.  Ensure you have **Fabric Loader** installed for Minecraft 1.17.1.
2.  Download the latest version of the mod.
3.  Place the mod `.jar` file into your Minecraft `mods` folder.
4.  Make sure you also have the **Fabric API** mod installed.

## Requirements

- Minecraft 1.17.1
- Fabric Loader >= 0.14.25
- Fabric API
- Java >= 16
